# InvoiceGuard
Hackathon Project-InvoiceGuard

## Steps to excute app
1-Set up gemini key and SMTP user,pwd, email sender and email receiver in .env file
2-To install required packages, run "pip install -r installation.txt" in terminal
3-Use a fresh terminal for starting fastAPI payment_api.py by running "python3 -m uvicorn payment_api:app --reload --port 8000" in terminal.
4-Open new terminal and run "python3 -m streamlit run main.py "to execute the application, then you will get the pop-up window below, click the assigned port (Open in Browser) which will navigate to streamlit application window.
